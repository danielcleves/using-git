using-git

The following commands are the most commonly used in Git

git init

start a repository

git status

see current status

git add <filename>

add a file to the staggin area

git add .

makes a git add of everything

git rm --cached <filename>

removes the copy of the file from the staging-area, without touching the working tree copy

git log <filename>

commit history

If a file name is included, it shows the history of changes to that file.

git log --stat

commit history with number of bytes that were changed

git show <filename>

graphically displays a file's change history
